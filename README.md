> ## 1. Vue基础
>> 1.1 Vue实例的作用范围是什么

Vue会管理el选项`命中的元素`及其内部的后代元素

>> 1.2 是否可以使用其他的选择器

可以使用其他的选择器，但是建议使用ID选择器

>> 1.3 是否可以设置其他的dom元素

可以使用其他的`双标签`，但是不能使用`HTML`和`BODY`

>> 1.4 el: 挂载点

时用来设置Vue实例挂载（管理）的元素

> ## 2. 本地应用
>> 2.1 内容绑定，事件绑定
### (1) V-text
* 设置标签的内容
* 默认写法会替换全部内容，使用差值表达式{{}}可以替换指定内容
### (1) V-html 
* 设置元素的innerHTML
* 内容中有html结构会被解析为标签
### (1) V-on
* 元素绑定事件
* 事件名不需要写on
* 指令可以简写为@
* 绑定的方法定义为再methods属性中
